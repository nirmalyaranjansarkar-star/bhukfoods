### How to Update the Website

1.  **Make Changes:** Edit your code in VS Code.
2.  **Test Locally:** 
    ```bash
    npm run dev
    ```
3.  **Deploy Manually:**
    ```bash
    npm run deploy
    ```
    This command compiles your TypeScript code, builds the static assets, and uploads them to Firebase Hosting.

### Local Development
To run the app on your machine:
```bash
npm install
npm run dev
```
